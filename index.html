This is code for the tomasina sanctuary website

Welcome to the development repository for tomasinasanctuary.org. This contains the code that runs the tomasinasanctuary.org and also contains a snapshot of the latest db

Dependencies

no dependencies.

Install

The website requires LAMP, Apache 2.2 PHP 5, Mysql 5. put the contents of the htdocs in your document root, create a database and import the tomasinasanctuary.org.sql.tgz. The htdocs contains a working phpmysql so you just need to browse to it and import using phpmysql. Then update configuration.php with the databse details.
